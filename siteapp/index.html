About SiteApp

We aim to build a website and a mobile app using discussion boards and gamification to make learning fun and accessible to ALL students while building community.

We are currently working on a web platform to host online learning games. Upcoming features include a profile section, leaderboard, forum to connect with other students and mentors, and a live chat for instant homework help!
